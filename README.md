# 🎓 Student Management Portal (C++)

A console-based **Student Management System** developed in **C++** for managing student records efficiently.
This project demonstrates the practical implementation of:

* Structures
* Functions
* Arrays
* Loops & Conditions
* Searching Algorithms
* Input Validation
* Menu Driven Programming

---

# 📌 Features

✅ Add New Students
✅ Delete Existing Students
✅ Search Student by Roll Number
✅ Display All Students Data
✅ Upgrade/Edit Student Information
✅ Scholarship Eligibility System
✅ Average Grade Calculation
✅ Password Protected Access
✅ Grade Validation & Uppercase Conversion

---

# 🚀 How to Run

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/student-management-portal.git
```

---

## 2️⃣ Open Project

Open the project in:

* Dev C++
* CodeBlocks
* Visual Studio
* VS Code

---

## 3️⃣ Compile

Using g++:

```bash
g++ main.cpp -o studentportal
```

---

## 4️⃣ Run

```bash
./studentportal
```

For Windows:

```bash
studentportal.exe
```
